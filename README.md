# La Cuisine - Restaurant Website

## Project Overview
La Cuisine is a responsive restaurant website built as part of a front-end development assignment. The project demonstrates modern web design practices using Bootstrap 5, HTML5, and CSS3.

## Features
- **Responsive Navigation** - Fixed navigation bar with mobile menu toggle
- **Hero Carousel** - Eye-catching carousel display for showcasing restaurant images
- **Menu Section** - Organized menu display
- **About Section** - Information about the restaurant
- **Contact Section** - Contact information and reservation details
- **Table Reservation** - Modal form for booking tables
- **Modern Design** - Clean, professional aesthetic with smooth transitions and hover effects
- **Responsive Layout** - Mobile-first design that works on all device sizes

## Technologies Used
- **HTML5** - Semantic markup structure
- **CSS3** - Custom styling with animations and transitions
- **Bootstrap 5** - CSS framework for responsive design
- **JavaScript** - Bootstrap's JavaScript for interactive components

## Getting Started
1. Open `index.html` in a web browser
2. Navigate through the website using the top navigation bar
3. Click "Book Table" to open the reservation modal
4. Explore different sections: Home, Menu, About, and Contact

## Browser Support
This website is compatible with modern browsers:
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Responsive Breakpoints
- Mobile: < 576px
- Tablet: ≥ 576px
- Desktop: ≥ 992px
